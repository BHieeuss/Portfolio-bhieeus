# Image Assets Required

Để website hoạt động hoàn chỉnh, bạn cần thêm các file ảnh sau:

## Avatar Image

- **File**: `assets/images/avatar.jpg`
- **Kích thước**: 400x400px (sẽ được resize về 120x120px)
- **Format**: JPG hoặc PNG
- **Mô tả**: Ảnh đại diện của bạn, nên là ảnh vuông

## Portfolio Images

Thêm 6 ảnh dự án vào thư mục `assets/images/portfolio/`:

- `project1.jpg` - Cyberpunk Runner (Unity Mobile Game)
- `project2.jpg` - Space Explorer (Unreal Engine 5 Game)
- `project3.jpg` - Fantasy RPG (Unity 3D Adventure)
- `project4.jpg` - Puzzle Master (Mobile Puzzle Game)
- `project5.jpg` - Character Models (3D Character Art)
- `project6.jpg` - VR Experience (Unreal VR Game)

**Specs cho Portfolio Images:**

- **Kích thước**: 500x300px
- **Format**: JPG
- **Chất lượng**: High quality, game screenshots hoặc 3D renders

## Blog Images

Thêm 3 ảnh blog vào thư mục `assets/images/blog/`:

- `blog1.jpg` - Advanced AI in Unity Games
- `blog2.jpg` - Optimizing Games for Mobile
- `blog3.jpg` - Unreal Engine 5 Nanite

**Specs cho Blog Images:**

- **Kích thước**: 400x250px
- **Format**: JPG
- **Nội dung**: Screenshots, diagrams, hoặc concept art liên quan đến game development

## Audio File (Optional)

- **File**: `assets/audio/cyberpunk-ambient.mp3`
- **Loại**: Ambient/Electronic music với phong cách cyberpunk
- **Thời lượng**: 2-5 phút (sẽ loop)
- **Format**: MP3
- **Chất lượng**: 128kbps hoặc cao hơn

## Gợi ý nguồn ảnh miễn phí:

- **Unsplash.com** - Ảnh chất lượng cao
- **Pexels.com** - Ảnh free stock
- **Pixabay.com** - Ảnh và vector miễn phí
- **Unity Asset Store** - Screenshots từ game demos
- **ArtStation.com** - 3D art và game art
- **Freepik.com** - Vector và mockups
